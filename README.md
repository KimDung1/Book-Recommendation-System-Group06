This project builds a Book Recommendation System using the **Goodbooks-10k** dataset, fulfilling the requirements for DS 423 — Machine Learning with Large Datasets.

## Setup Instructions
1. **Prerequisites:** 
   - Install Python 3.8+
   - Jupyter Notebook or JupyterLab
2. **Kaggle API:**
   - Place your `kaggle.json` (API Token) in this directory or in `~/.kaggle/` to allow automatic dataset downloading.
3. **Execution:**
   - Open `Group06_BookRecommendation_LeKimDung.ipynb`.
   - Run the first cell to install all required libraries (`pandas`, `matplotlib`, `seaborn`, `scikit-learn`, `kaggle`).
   - Run the notebook sequentially. It will download the Goodbooks-10k dataset, perform Data Cleaning, EDA, Feature Engineering, and save the processed data to the `data/` folder for Modeling.

## Files
- `Group06_BookRecommendation_LeKimDung.ipynb`: The primary notebook for Data Collection, Cleaning, EDA, and Feature Engineering.
- `data/`: Folder that will be automatically generated containing the datasets.
- `report_template.tex`: LaTeX template for the final report.
